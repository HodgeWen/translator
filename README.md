# 星禾

<div align="center">中文 · <a href="./README.en.md">English</a></div>

基于大语言模型的浏览器翻译扩展，支持任意兼容 OpenAI API 的服务商。提供四种翻译方式和灵活的翻译服务配置。

## 功能

- **页面全文翻译** — Alt+W 一键翻译整页，自动感知页面路由切换
- **悬浮翻译** — 按住快捷键（默认 Ctrl）并指向段落即可高亮翻译
- **输入框翻译** — 在输入框中连按三次快捷键翻译内容，再按三次还原
- **弹窗翻译** — 手动输入文本翻译，支持一词多义展示和语气切换
- **翻译服务** — Single（单服务商 + 模型自动降级）或 Pool（多服务商加权负载均衡）
- **四种翻译样式** — 原文替换 / 双语对照 / 下划线（鼠标悬停查看原文）/ 纯净替换
- **级联配置** — 展示样式与翻译语气三级级联：服务默认 → 个人默认 → 全局覆盖
- **行内 HTML 保留** — 代码、链接、强调文字等元素翻译后无损保留
- **聚合翻译** — 多个段落合并为一次请求，减少调用次数，失败自动回落逐段翻译
- **翻译缓存** — 本地缓存，7 天自动清理，修改提示词后旧缓存自动失效，相同请求智能合并
- **语言检测** — 内置本地检测，可选 Google 或自定义 API 检测器备用
- **自定义语气与预设** — 自定义翻译风格和可复用的提示词模板
- **加载动画** — 四种主题 + 可自定义的输入框动画
- **加密导出** — 设置导出为 JSON，支持密码加密保护 API Key
- **双语言界面** — 中文 / English
- **网站规则** — 针对 GitHub 等站点的翻译排除

## 架构

```
src/
├── entrypoints/
│   ├── background.ts          # 后台模块：消息路由、缓存管理、快捷键
│   ├── content/               # 内容脚本：段落识别→语言检测→编码→请求→结果应用
│   ├── popup/main.tsx         # 弹窗翻译界面
│   └── options/main.tsx       # 设置页（8 标签页）
├── lib/                       # 核心库
│   ├── api/                   # 翻译 API 调用、服务队列、流式处理
│   ├── storage.ts, cache.ts   # 配置存储与翻译缓存
│   ├── schema.ts, prompts.ts  # 校验与提示词模板
│   ├── lang-detect, block-detect, inline-placeholder, batch-protocol
│   └── messaging, crypto, site-rules
├── components/                # React 组件
└── types/                     # 类型定义
```

## 安装

```bash
bun install && bun build

```

## 设置概览

| 标签页     | 说明                                       |
| ---------- | ------------------------------------------ |
| 服务商     | 管理翻译 API 提供商（地址、Key、模型等）   |
| 翻译服务   | 配置 Single（+降级）或 Pool（加权）服务     |
| 预设与风格 | 自定义提示词预设和翻译语气风格             |
| 语言       | 母语、源语言、界面语言、检测器             |
| 显示       | 展示样式、翻译语气、加载动画               |
| 通用       | 全局提示词、请求超时、聚合参数、快捷键     |
| 备份与恢复 | 设置导出/导入（支持加密）                  |
| 使用手册   | 完整使用文档                               |

## 权限

`storage` `activeTab` `scripting` `alarms` `host_permissions: <all_urls>`

## 隐私

翻译请求直连你配置的服务商，不经过任何中间服务器。API Key 仅保存在浏览器中。导出支持加密保护。无数据收集。
