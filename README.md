# 智能翻译器

[English](./README.en.md)

基于 OpenAI 兼容大模型的浏览器翻译扩展，支持多种翻译服务商接入、四种页面翻译样式、快捷弹窗翻译。

## 功能

- **多服务商支持** — 支持任意 OpenAI-compatible API，可同时配置多个服务商，模型队列自动降级
- **四种翻译样式** — 原文替换 / 双语对照 / 下划线（hover 显示原文） / 纯净替换
- **页面实时翻译** — 可视区域翻译，聚合批量请求，提高效率
- **弹窗快速翻译** — 工具栏 popup 支持划词翻译、语言自动检测
- **输入框翻译** — 连续输入三个空格即可翻译当前输入框内容
- **Ctrl + Hover 翻译** — 按住 Ctrl 悬停段落即可高亮翻译
- **语言检测** — 内置 franc-min，可选 Google / 自定义 API 检测器备用
- **SPA 路由感知** — 自动感知 pushState / replaceState 路由变化，重新翻译新页面内容
- **翻译缓存** — IndexedDB 缓存翻译结果，每日自动清理过期条目
- **设置导入导出** — JSON 导出，可选 AES-GCM 加密保护 API Key
- **快捷键** — `Alt+W` 切换页面翻译
- **i18n** — 支持中文 / English 界面语言切换

### 翻译样式

| 样式 | 描述 |
|---|---|
| 原文替换 | 替换文本但保留元素结构 |
| 双语对照 | 原文与译文同时显示 |
| 下划线 | 下划线显示译文，鼠标悬停查看原文 |
| 纯净替换 | 无额外标记的纯净替换 |

## 权限说明

| 权限 | 用途 |
|---|---|
| `storage` | 保存用户设置 |
| `activeTab` | 获取当前标签页信息 |
| `scripting` | 注入翻译脚本 |
| `alarms` | 定时清理翻译缓存 |
| `host_permissions: <all_urls>` | 在任意网页上支持用户发起的翻译操作 |

## 隐私

- 翻译请求直接发送到用户配置的服务商 API，不经过任何中间服务器
- API Key 存储在 chrome.storage.sync 中，导出时支持 AES-GCM 加密
- 浏览器本地 IndexedDB 缓存翻译结果，定时自动清理
- 不收集任何用户数据，不上报任何遥测
